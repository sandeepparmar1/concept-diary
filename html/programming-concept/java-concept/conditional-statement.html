Conditional Statements

Control the flow of your program using decisions.

Introduction

Conditional statements allow your program to execute certain blocks of code only when a condition is true. Each condition evaluates to a boolean value (true or false).

1. if Statement

if (condition) {
    // statements
}
  • If the condition is true, the code executes.
  • Condition can be a boolean expression or a boolean variable.
if (2 == 2) {
    System.out.println("True!");
}

int i = 10;
if (i > 9) {
    System.out.println("i is greater than 9");
}

2. if–else Statement

if (condition) {
    // executes if true
} else {
    // executes if false
}
int marks = 80;

if (marks >= 50) {
    System.out.println("Pass");
} else {
    System.out.println("Fail");
}

3. if–else if–else Ladder

if (condition1) {
    // block 1
} else if (condition2) {
    // block 2
} else {
    // default block
}
int marks = 85;

if (marks >= 90) {
    System.out.println("Grade A");
} else if (marks >= 75) {
    System.out.println("Grade B");
} else if (marks >= 60) {
    System.out.println("Grade C");
} else {
    System.out.println("Fail");
}

4. Variable Initialization Inside if

int i = 10;
int j;
if (i == 10) {
    j = 20;
}
System.out.println(j); // ❌ Compile-time error

Error: variable j might not have been initialized

Local variables must be initialized before use. The compiler cannot guarantee that j was assigned, as the condition might be false.

Fixed Version

int i = 10;
int j;
if (i == 10) {
    j = 20;
} else {
    j = 30;
}
System.out.println(j); // ✅ Works fine

Constant Expression Case

final int i = 10;
int j;
if (i == 10) {
    j = 20;
}
System.out.println(j); // ✅ Works fine

5.. switch Statement

switch (expression) {
    case value1:
        // statements
        break;
    default:
        // statements
}
  • Expression can be byte, short, int, char, or String.
  • Cannot use long, float, double, or boolean.
  • If break is missing, execution “falls through”.
int day = 3;

switch (day) {
    case 1:
        System.out.println("Monday");
        break;
    case 2:
        System.out.println("Tuesday");
        break;
    case 3:
        System.out.println("Wednesday");
        break;
    default:
        System.out.println("Invalid day");
}

Output: Wednesday
